This is a pastebin made by Yasoob Khalid as a side project.

I made it as a small project just to learn the skills required to make a pastebin and to learn a little about databases and user authentication.

This pastebin is not currently OpenSourced but I will try to release the source code soon on GitHub. This pastebin offers some unique options. Here are some of the features:

  • You can choose to post anonymous pastes (If you are logged out then your post is anonymous)
  • Automatic focus on textarea so that you can quickly start typing
  • Syntax highlighting
  • Signup and keep track of your pastes
  • You can delete your paste anytime if you have signed up
  • Commenting system
